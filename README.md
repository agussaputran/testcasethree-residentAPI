# Simple REST API with GO-GIN
Simple REST API for resident

## Tech Stack :
- Go 1.15
- Gin Gonic Framework
- PostgreSQL
- Gorm
- JWT
- Sentry
- Fetch API from rajaongkir.com for Province and City data
